# 🕹️ Jokenpô - DevClub

Este é um projeto clássico de "Pedra, Papel e Tesoura" desenvolvido como parte da minha jornada de aprendizado no **DevClub**. O objetivo principal foi consolidar conhecimentos em **JavaScript Vanilla**, manipulação do **DOM** e estilização avançada com **CSS**.

---

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do jogo.
* **CSS3:** Layout responsivo (Flexbox), animações com `@keyframes` e efeitos de transição.
* **JavaScript:** Lógica de jogo, geração de escolhas aleatórias e atualização dinâmica da interface.

---

## 🛠️ Funcionalidades

* **Jogo contra a máquina:** Lógica automatizada para escolha do computador.
* **Placar em tempo real:** Atualização instantânea de vitórias, derrotas e empates.
* **Design Responsivo:** Interface adaptada para dispositivos móveis e desktop.
* **Feedback Visual:** Animações de *shake* nos botões e sistema de cores para facilitar a leitura do resultado.
* **Sistema de Reset:** Função para zerar o progresso sem recarregar a página.

---

## 🧠 Desafios e Aprendizados

Durante o desenvolvimento, foquei em resolver problemas reais de interface:
1.  **Estabilidade de Layout:** Utilizei `min-height` e `min-width` para evitar o *Layout Shift* (saltos na tela) quando o texto do resultado aparecia.
2.  **Lógica Escalável:** A função do computador foi estruturada para ser dinâmica, facilitando a adição de novas opções (como o "Lizard" ou "Spock") no futuro.
3.  **Experiência do Usuário (UX):** Implementei animações suaves e cursor diferenciado para indicar elementos interativos.
